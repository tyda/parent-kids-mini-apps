# 安靜聲音獵人

不用奔跑、不用大聲的親子聽覺觀察小遊戲。適合車上、餐廳、排隊、睡前等 5–10 分鐘空檔。

## 玩法

1. 選擇現在的地點。
2. 抽一張任務卡。
3. 安靜聽 20 秒，找符合卡片提示的聲音。
4. 輪流描述聽到的聲音、猜來源，完成卡片上的小任務。
5. 完成後收集一顆星，最多 5 顆。

## 檔案

- `index.html`：遊戲畫面
- `styles.css`：手機與橫向螢幕版面
- `app.js`：抽卡、計時、星星邏輯
- `test-core.js`：核心邏輯測試
- `manifest.webmanifest` / `sw.js`：PWA 與弱網路快取基礎

## 驗證

```bash
node test-core.js
```
