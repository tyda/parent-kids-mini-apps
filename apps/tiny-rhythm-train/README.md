# 手指節奏小火車

安靜、低壓力的親子節奏小遊戲。適合車上、餐廳、排隊、睡前 5–10 分鐘互動。

## 玩法

1. 按「開一班新列車」抽出節奏、車站和小任務。
2. 大人或孩子當列車長，用手指在桌面、膝蓋或手掌上輕敲節奏。
3. 另一位乘客模仿一次，完成小任務後按「抵達下一站」。
4. 收集 3 個車站就完成一趟旅程，可以換人當列車長。

## 設計重點

- 不需要登入、不需要道具。
- 在安靜場域可以完全用手指動作完成。
- 手機直向與橫向都可使用。
- PWA 基本快取，弱網路下仍可重開。
