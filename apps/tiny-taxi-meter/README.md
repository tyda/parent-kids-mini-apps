# 小小計程車行｜親子計程車跳錶 🚕

一個手機可直接開啟的親子遊戲版計程車跳錶網頁。

## 功能

- 起程運價：前 1.25 公里 85 元
- 續程運價：每 200 公尺跳 5 元
- 延滯 / 等待計時：每 60 秒或 100 秒累計 5 元
- 按鈕：開始、暫停、重置、切換夜間加成 +20%、等待開/關、GPS開/關、手動加距離、吃零食 +100m、看影片 +500m
- 手機友善大字體介面
- 可離線快取基本檔案（瀏覽器支援 service worker 時）

## 開啟方式

### 方式一：直接開檔案

在手機或電腦瀏覽器開啟：

`index.html`

注意：部分手機瀏覽器直接開本機檔案時，GPS 或離線快取可能受限。

### 方式二：用簡易 HTTP Server（推薦）

在此資料夾執行：

```bash
python -m http.server 8080
```

再用手機連到同一網路下的電腦 IP：

```text
http://電腦IP:8080/
```

若要使用 GPS，正式環境建議用 HTTPS；localhost 通常也可用。

## 測試

### Node 核心邏輯測試

```bash
node test-core.js
```

### 瀏覽器測試頁

開啟：

`tests.html`

## 安全提醒

- 這是親子遊戲，不是正式計程車計價器。
- 在車上玩時，請由乘客或小朋友操作，駕駛不要操作手機。
- GPS 在車內可能受到玻璃、建築物、隧道影響；若跳動不準，可以用「遊戲加 200m / 1km」、「吃零食 +100m」、「看影片 +500m」手動模擬。
