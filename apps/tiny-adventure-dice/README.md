# 親子任務骰子

一個手機友善、可離線快取的親子小遊戲。

## 玩法

按「抽一題」會抽出：

- 想像場景：只是故事背景，不用真的到那裡
- 小任務：孩子實際要做的小動作或觀察題
- 完成獎勵：完成後可以得到的小鼓勵

孩子完成任務後按「完成 +1」累積星星，5 顆星可以換一個小獎勵。

## 適合情境

- 車上不要太吵鬧時
- 餐廳等餐
- 睡前 5 分鐘
- 週末出門前暖身

## 本機執行

```bash
cd /data/user-data/tiny-adventure-dice
python -m http.server 8081 --bind 127.0.0.1
```

打開：

```text
http://127.0.0.1:8081/
```
